# Name

## Overview

tail コマンドを python で実装しました。

## Description

ファイルの末尾から n 行を出力します。

## Usage

`pytail -n 10 -f ./sample.txt`

### option

- -n 行末から出力する行数を数値で指定
- -f ファイルパスを指定

## Install

`pip install git+https://github.com/naokikomiya/tail_command_by_python.git`
